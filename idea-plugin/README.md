# AI File CleanKeeper — IntelliJ IDEA 插件

智能识别 AI 生成文件、临时文件与配置文件，导入后自动清理或加入 Ignore，保持本地工作区干净。

## 功能

| 能力 | 说明 |
| --- | --- |
| **自动扫描** | 文件导入、复制、移动时自动触发分类 |
| **规则 + AI 双引擎** | 内置启发式规则快速识别；可选 OpenAI 兼容 API 深度判断 |
| **临时文件清理** | `.tmp`、`.bak`、`~`、`Thumbs.db` 等自动删除 |
| **AI 无用产物清理** | 识别 AI 草稿、脚手架、示例输出并清理 |
| **配置类文件 Ignore** | `.env`、`.cursorrules`、`AGENTS.md` 等自动写入 `.gitignore` |
| **可疑文件人工处置** | 底部工具窗口 + 通知栏支持 **隔离转存** / **删除** / **加入 Ignore** |
| **可配置** | API Key、Base URL、Model、隔离目录、自动策略开关 |

## 安装

### 方式一：从源码构建

```bash
cd idea-plugin
./gradlew buildPlugin
```

构建产物位于 `build/distributions/ai-file-cleankeeper-1.0.0.zip`。

在 IDEA 中：**Settings → Plugins → ⚙ → Install Plugin from Disk** → 选择 zip 文件。

### 方式二：Run IDE（开发调试）

```bash
cd idea-plugin
./gradlew runIde
```

## 配置

**Settings → Tools → AI File CleanKeeper**

| 配置项 | 说明 | 默认值 |
| --- | --- | --- |
| API Key | OpenAI 兼容接口密钥 | 空 |
| Base URL | API 地址 | `https://api.openai.com/v1` |
| Model | 模型名称 | `gpt-4o-mini` |
| 隔离目录 | 可疑文件转存路径（相对项目根） | `.cleankeeper/quarantine` |
| 自动删除临时文件 | 检测到 tmp 文件立即删除 | 开启 |
| 自动删除 AI 无用产物 | 检测到无用 AI 文件立即删除 | 开启 |
| 自动加入 Ignore | 配置类文件写入 `.gitignore` | 开启 |
| 启用 AI 辅助分类 | 调用 API 进行深度判断 | 开启 |

支持 DeepSeek、通义千问、Moonshot 等任意 OpenAI 兼容服务，只需修改 Base URL 和 Model。

## 使用

### 自动模式（默认）

打开项目后，插件监听 VFS 变更。新导入的文件会：

1. 先用内置规则快速分类
2. 规则不确定时调用 AI API（如已配置）
3. 按分类执行自动清理或 Ignore

### 手动操作

- **Tools → AI CleanKeeper → 扫描整个项目**
- 项目树右键 → **AI CleanKeeper** → 扫描 / 隔离 / 删除 / 加入 Ignore
- 底部 **AI CleanKeeper** 工具窗口查看所有待处理文件

### 文件分类

| 分类 | 处理方式 |
| --- | --- |
| 临时文件 | 自动删除（可关闭） |
| AI 无用产物 | 自动删除（可关闭） |
| 项目配置 | 自动加入 `.gitignore` |
| AI 配置 | 自动加入 `.gitignore` |
| 可疑文件 | 通知 + 工具窗口，等待人工操作 |
| 安全 | 不处理 |

## 内置识别规则（部分）

**临时文件：** `*.tmp`, `*.bak`, `*~`, `.swp`, `Thumbs.db`, `.DS_Store`

**AI 配置：** `.cursorrules`, `.cursorignore`, `AGENTS.md`, `.cursor/`, `.aider*`

**项目配置：** `.env`, `.env.local`, `local.properties`, `secrets.yml`

**AI 产物：** 文件名含 `draft_`, `generated_`, `ai_output`；内容含 `generated by ai` 等标记

可在设置中扩展自定义规则模式。

## 兼容性

- IntelliJ IDEA 2023.3+
- 也适用于 Android Studio、PyCharm 等 JetBrains IDE（Platform 插件）

## 注意事项

- 自动删除操作不可撤销，建议先开启通知确认，熟悉后再完全自动化
- AI 分类需要网络访问 Base URL，未配置 API 时仅使用规则引擎
- 隔离目录 `.cleankeeper/quarantine` 建议加入 `.gitignore`
